**客户：** 帧龙虾  
**创建日期：** 2026-03-14  
**状态：** 第一版计划完成

---

## 核心功能流程

1. 多平台数据抓取分析热点话题
2. LLM 大模型生成文案脚本
3. 调用私有素材库，自动匹配画面音频
4. 确认素材，混剪视频
5. 定时自动上传视频
6. 自动检测数据，分析优化创意

---

## 系统架构（6 层）

```
热点感知层 → 数据分析层 → 内容生成层 → 素材管理层 → 视频生产层 → 发布运营层
```

---

## 6 大功能模块

### 1. 热点感知系统
- TrendRadar 对接 (P0)
- 智小红/智小抖 API (P0)
- 巨量算数爬取 (P1)
- 小红书蒲公英 (P1)
- 热点去重合并 (P0)

### 2. 爆款分析引擎
- 标题模板、开头钩子、文案结构
- 视频时长、分镜节奏、BGM 类型
- 发布时间、平台分析

### 3. 内容生成系统
- **LLM 策略：** Kimi 为主（成本低 + 中文强），Claude 为辅（核心创意）
- 生成：爆款标题、文案脚本、分镜描述、BGM 推荐

### 4. 素材管理系统
- 存储：MinIO（对象存储）
- 检索：Elasticsearch（标签化搜索）
- 处理：OpenCV + OCR
- 分类：视频/音频/图片/文案素材库

### 5. 视频生产系统
- 素材匹配：语义检索
- 视频混剪：FFmpeg
- 字幕生成：Whisper/OCR
- 封面生成：OpenCV
- 质量检查：人工确认

### 6. 发布运营系统
- 定时上传、多平台分发
- 数据监控（播放/点赞/评论/转发）
- 效果分析：Metabase 可视化
- 优化建议：A/B 测试 + 数据反馈

## Skill 化架构（OpenClaw）

所有功能模块 → 独立 Skill → OpenClaw 统一调度

**Skill 列表：**
- skill-hotspot-fetch（热点抓取）
- skill-hotspot-analyze（热点分析）
- skill-copywrite-gen（文案生成）
- skill-material-search（素材检索）
- skill-material-ocr（OCR 识别）
- skill-video-edit（视频混剪）
- skill-cover-gen（封面生成）
- skill-platform-publish（平台发布）
- skill-data-monitor（数据监控）
- skill-optimize-suggest（优化建议）

## 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| 平台 API 限制 | 高 | 准备多账号 + 爬虫备用 |
| 素材版权 | 高 | 优先 Pexels 等免版权源 |
| 内容同质化 | 中 | 人工审核 + 创意注入 |
| LLM 输出不稳定 | 中 | 多模型冗余 + 人工校验 |
| 视频质量波动 | 中 | 建立质量检查清单 |
