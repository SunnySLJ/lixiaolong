# 智小红 - 小红书自动化采集分析系统

## 📋 功能说明

本系统通过浏览器自动化技术，自动采集小红书热点数据，并进行深度分析。

**核心特点：**
- ✅ 浏览器自动化（模拟真人操作）
- ✅ 智能时间间隔控制（防风控）
- ✅ 自动导出多格式数据（JSON/Excel/CSV）
- ✅ 自动生成分析报告
- ✅ 安全频率控制

---

## 🚀 快速开始

### 方式 1：一键运行（推荐）

双击运行：
```
run_xhs_collector.bat
```

### 方式 2：分步运行

**步骤 1：安装依赖**
```bash
pip install playwright pandas openpyxl
playwright install chromium
```

**步骤 2：采集数据**
```bash
python xiaohongshu_collector.py
```

**步骤 3：分析数据**
```bash
python xhs_analyzer.py
```


## 📊 输出文件

采集完成后，在 `xhs_data/` 目录下生成：

| 文件 | 格式 | 说明 |
|------|------|------|
| `xhs_data_YYYYMMDD_HHMMSS.json` | JSON | 原始数据 |
| `xhs_data_YYYYMMDD_HHMMSS.xlsx` | Excel | 表格数据 |
| `xhs_data_YYYYMMDD_HHMMSS.csv` | CSV | 逗号分隔 |
| `analysis_report_*.txt` | 文本 | 详细分析报告 |
| `analysis_summary_*.json` | JSON | 数据摘要 |

## ⚠️ 安全提醒

### 频率限制（重要！）

| 操作 | 建议频率 | 极限频率 |
|------|---------|---------|
| 浏览笔记 | <10 条/分钟 | <20 条/分钟 |
| 查看详细数据 | <10 条/小时 | <50 条/小时 |
| 下载图片/视频 | <10 个/小时 | <50 个/小时 |

### 安全建议

1. ✅ **使用小号操作** - 避免主账号风险
2. ✅ **分时操作** - 不要一次性采集太多
3. ✅ **模拟真人** - 脚本已设置 slow_mo
4. ✅ **遵守规则** - 不要违反平台规定

### 风险信号

如果出现以下情况，立即停止：
- ❌ 需要验证码
- ❌ 账号被限制
- ❌ 页面加载异常
- ❌ 数据无法获取

## 📞 常见问题

### Q: 浏览器打不开？
A: 运行 `playwright install chromium` 安装浏览器

### Q: 登录不了？
A: 第一次需要手动登录，之后会保存 Cookie

### Q: 数据采集不到？
A: 检查网络连接，或小红书页面结构是否更新

### Q: 被封号了怎么办？
A: 降低采集频率，使用小号，分时操作
