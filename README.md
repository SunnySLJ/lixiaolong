# 视频自动化生产系统 - 帧龙虾

🎯 基于 OpenClaw 的视频内容自动化生产平台

## 项目概述

本项目是一个完整的视频自动化生产系统，从热点抓取到视频发布的全流程自动化。

### 核心功能

1. **热点感知** - 多平台数据抓取分析热点话题
2. **内容生成** - LLM 大模型生成文案脚本
3. **素材匹配** - 调用私有素材库，自动匹配画面音频
4. **视频混剪** - 确认素材，FFmpeg 混剪视频
5. **定时发布** - 自动上传视频到多平台
6. **数据分析** - 自动检测数据，分析优化创意

## 技术栈

### 核心框架
- **OpenClaw** - Agent 调度框架
- **Node.js** - 主要运行时

### AI 模型
- **Kimi** - 主力文案生成（成本低 + 中文强）
- **Claude** - 核心创意生成
- **OpenAI/Gemini/Qwen** - 备用方案

### 数据处理
- **FFmpeg** - 视频混剪
- **OpenCV** - 图像处理
- **Whisper** - 语音识别
- **Tesseract** - OCR 文字识别

### 存储
- **MinIO** - 对象存储
- **Elasticsearch** - 素材检索

### 数据可视化
- **Metabase** - 数据看板

## 快速开始

### 环境要求

- Node.js >= 18
- OpenClaw >= 1.0
- FFmpeg
- Python 3.9+ (部分脚本需要)

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/SunnySLJ/lixiaolong.git
cd lixiaolong

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 API 密钥

# 运行第一个 Skill
openclaw skill run skill-hotspot-fetch
```

## 风险与应对

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| 平台 API 限制 | 高 | 准备多账号 + 爬虫备用 |
| 素材版权 | 高 | 优先 Pexels 等免版权源 |
| 内容同质化 | 中 | 人工审核 + 创意注入 |
| LLM 输出不稳定 | 中 | 多模型冗余 + 人工校验 |
| 视频质量波动 | 中 | 建立质量检查清单 |
