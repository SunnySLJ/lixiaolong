## 📋 技能说明

这是一个专业的前端设计技能，用于创建**独特、生产级的前端界面**，避免通用的"AI 风格"设计。

### 何时使用

- 构建网页组件、页面或应用
- 创建落地页、仪表盘或网站
- 设计 React/Vue 组件
- 美化任何 Web UI
- 制作海报、艺术品或视觉界面

---

## 🎯 核心设计理念

### 1. 大胆的美学方向

在使用此技能前，明确以下四点：

**目的（Purpose）**
- 这个界面解决什么问题？
- 目标用户是谁？
- 情感目标是什么？

**基调（Tone）**
选择一个极端的美学方向：
- 极简主义
- 极繁主义
- 复古未来
- 有机自然
- 奢华精致
-  playful 趣味
- 杂志编辑风
- 粗野主义
- 装饰艺术
- 柔和粉彩
- 工业 utilitarian

**关键：** 选择清晰的概念方向并精确执行。大胆极繁和精致极简都有效 - 关键是意向性，不是强度。

**差异化（Differentiation）**
- 什么让这个设计令人难忘？
- 用户会记住的一个特点是什么？
- 如何从通用设计中脱颖而出？

### 2. 实现标准

代码必须是：
- ✅ 生产级且功能完整
- ✅ 视觉冲击力强且令人难忘
- ✅ 具有清晰美学观点的 cohesive 设计
- ✅ 每个细节都精心打磨

## 🎨 美学指南

### 排版 Typography
❌ 避免：Arial, Inter, Roboto, 系统字体
✅ 推荐：独特的展示字体 + 精致的正文字体配对
✅ 资源：Google Fonts, Adobe Fonts, 定制字体

### 颜色与主题
❌ 避免：白色背景上的紫色渐变（AI 标配）
✅ 推荐：
- 主色调 + 锐利强调色
- 使用 CSS 变量保持一致性
- 为每个项目创建定制调色板

### 动效 Motion
✅ 推荐：
- CSS-only 解决方案（HTML 项目）
- Motion 库（React 项目）
- 精心编排的页面加载（staggered reveals）
- 滚动触发动效
- 惊喜的 hover 状态

### 空间构成
✅ 推荐：
- 意想不到的布局
- 不对称和重叠
- 对角线流
- 打破网格的元素
- 大量留白 OR 控制密度

### 背景与视觉细节
✅ 推荐：
- 渐变网格
- 噪点纹理
- 几何图案
- 分层透明
- 戏剧性阴影
- 装饰边框
- 定制光标
- 颗粒叠加

---

## ❌ 绝对避免

- Inter, Roboto, Arial 等过度使用字体
- 白底紫色渐变（AI 标配）
- 可预测的布局和组件模式
- 缺乏上下文特色的模板化设计
- Space Grotesq 作为默认"独特"字体
- 居中文本的通用 hero 区域
- Bootstrap 外观组件
- 未定制的 Tailwind 默认样式

## 📊 设计复杂度匹配

**重要：** 实现复杂度要与美学愿景匹配

### 极繁设计
需要：
- 精心编写的代码
- 大量动画和效果
- 多层次视觉元素
- 复杂的交互

### 极简/精致设计
需要：
- 克制和精确
- 对间距、排版、细节的关注
- 微妙的细节处理
- 完美的执行
