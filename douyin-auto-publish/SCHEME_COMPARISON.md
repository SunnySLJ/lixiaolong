## 🎯 两种技术方案对比

### 方案 A：官方 API（推荐用于企业）

**文档**: https://open.douyin.com/platform/resource/docs/ability/content-management/douyin-publish-solution/

#### ✅ 优点

- 官方支持，稳定可靠
- 不易被封号或触发风控
- 功能完整（支持话题、@用户、POI、小程序等）
- 可以管理已发布视频（查看数据、删除等）

#### ❌ 缺点

- **需要企业资质**（个人无法申请）
- 需要 OAuth 用户授权流程
- 申请周期长（审核 1-2 周）
- 需要 client_key 和 client_secret

#### 📋 申请条件

1. ✅ 企业营业执照或 MCN 机构资质
2. ✅ 通过字节跳动开放平台认证
3. ✅ 创建应用并审核通过
4. ✅ 获取 client_key 和 client_secret

#### 🔧 实现流程

```
用户授权 (OAuth) 
  → 获取 access_token 
  → 上传视频 
  → 获取 video_id 
  → 调用发布接口
```

#### 💻 核心接口

```python
# 1. 上传视频
POST https://open.douyin.com/api/douyin/v1/video/upload_video/

# 2. 发布视频
POST https://open.douyin.com/api/douyin/v1/video/create/
```

---

### 方案 B：网页自动化（适合个人）

**当前实现**: Playwright 模拟用户操作

#### ✅ 优点

- 个人用户可用
- 无需审核
- 快速开发和部署
- 零成本

#### ❌ 缺点

- 可能触发抖音风控
- 页面更新需要维护
- 稳定性不如官方 API
- 有封号风险（频繁操作）

#### 🔧 实现流程

```
扫码登录 
  → 保存 Cookie 
  → 模拟上传 
  → 填写信息 
  → 点击发布
```

## ❓ 常见问题

### Q1: 个人能申请官方 API 吗？

**A**: 不能。官方 API 需要企业资质，个人账号无法申请。

### Q2: 网页自动化会被封号吗？

**A**: 有可能。如果频繁操作或触发风控，可能被封号。建议：
- 控制发布频率（< 5 条/天）
- 使用真实网络环境
- 不要使用代理
- 定期更新登录状态

### Q3: 官方 API 收费吗？

**A**: 目前免费。但需要企业资质。

### Q4: 两种方案可以同时使用吗？

**A**: 可以。建议：
- 主账号使用官方 API（稳定）
- 小号使用网页自动化（灵活）
