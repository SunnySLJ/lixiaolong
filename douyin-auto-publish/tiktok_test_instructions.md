
# TikTok 上传测试脚本

在支持 MCP 的 AI 客户端中运行以下命令：

## 1. 检查连接

```
检查我的 TikTok 连接状态
```

## 2. 连接 TikTok（如未连接）

```
帮我连接 TikTok 账号
```

## 3. 上传视频

```
帮我上传这个视频到 TikTok: ./storage/videos/test.mp4
标题："AI 自动化测试视频 #AI #自动化 #科技"
隐私级别：公开
```

## 4. 查看发布状态

```
帮我检查刚才发布的视频状态
```

## 5. 查看我的视频列表

```
帮我查看我发布的视频列表
```
